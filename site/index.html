KMediaBridge

Public, version-preserving Maven repository for the independently replaceable LGPL native runtime.

repositories {
    maven("https://shusek.github.io/KMediaBridge/maven")
}

dependencies {
    runtimeOnly("io.github.shusek:kmedia-bridge-ffmpeg-runtime-desktop:0.4.0")
}

The internal-use API and Kotlin backend are not published here. Historical versions through 0.3.0 retain the LGPL terms conveyed with those releases.
